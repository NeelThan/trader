# Architecture Decision Records

This folder contains Architecture Decision Records (ADRs) for the Trader project.

## What is an ADR?

An Architecture Decision Record captures an important architectural decision made along with its context and consequences.

## How to use

### Create a new ADR

```bash
npm run adr:new
```

### Preview ADRs locally

```bash
npm run adr:preview
```

### Build static site

```bash
npm run adr:build
```

## ADR Template

See [template.md](./template.md) for the ADR template based on the MADR format.
